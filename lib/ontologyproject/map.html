Ontology Visualization Interface

Instructions:

  • Select an Ontology version from the top dropdown menu
  • Select a zoom value.
    • 'Zoom in' is easier to read, with larger images and larger text
    • 'Zoom out' shows more of the map, but is harder to read.
  • Select a Theme
    • 'Building 3D' uses animated images to represent buildings
    • 'Building 2D' uses 2D shapes to represent buildings
    • 'Station icons' displays the ontology as a subway map
  • Click the 'Visualize!' button to load the ontology visualization.
