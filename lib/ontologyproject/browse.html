Shopping Ontology

User Interface Instructions

  1. Enter an ontology term such as 'NamedShop' into the search box in the bottom left.
  2. A list of related classes will then appear in the Navigation Bar box in the above.
  3. Classes within the navigation box can be selected for further exploration of the ontology.
  4. Upon clicking a concept in the Navigation Box, Direct Individuals of the class will be presented in the neighboring Direct Individuals Box.
  5. The tree view provided in the bottom right provides a hierarchical structure of the ontology.

Project Information

The shopping ontology was created using Protege Ontology Editor, and has been visualized using jOWL - a jQuery plugin for navigating and visualising OWL-RDFS documents.
